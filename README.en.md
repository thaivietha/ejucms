# ejucms（易居房产CMS）

#### Description
易居房产系统(ejucms)是一套本地化O2O房产网站平台系统。通过系统用户快速找到自己合适的房源
房产代理公司主推自己的独家签售的楼盘，及展示公司合作销售的楼盘，快速获取潜在的意向客户报名，促进成交。产品实现PC,手机数据同步，方便新房代理机构，租房公寓等互联网化扩张发展，帮助客户快速搭建类似于海南品房阁，楼86，房多多，Q房网， 麦田房产，房天下，海外房产等网站平台。
